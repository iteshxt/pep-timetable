# 📅 PEP Timetable

A beautiful, responsive, and installable Progressive Web App (PWA) designed to track, schedule, and compare daily timetables. Crafted with soft, desaturated pastel themes and glassmorphism.

---

## ✨ Key Features

- ⚡ **Live Status Tracker**: A dynamic status dashboard banner that indicates classes currently in session, upcoming lunch breaks, or free periods.
- 👥 **Dual Schedule Comparison**: Side-by-side visual comparison of multiple schedules on desktop, consolidated into a single card on mobile to prevent time redundancy.
- 🎨 **Harmonious Color Systems**: Tailored, desaturated color palettes (Sky Blue for DSA, Lavender/Indigo for Verbal) that adjust gracefully between Light and Dark themes.
- 📱 **Installable Web App (PWA)**: Full offline functionality powered by local service workers, and standalone layout configurations optimized for iOS & Android home screens.
- 🔔 **Smart Notifications**: Sends automated browser-native reminders 10 minutes before any scheduled class starts.

---

## 🛠️ Technology Stack

- **Core Framework**: [Next.js (App Router)](https://nextjs.org/) & TypeScript
- **Styling**: Modern Vanilla CSS with CSS Variables & Glassmorphic filters
- **Offline Capabilities**: Vanilla JavaScript Service Worker (PWA)
- **Deployment & SEO**: Native App Router Metadata & Apple Standalone Meta Tags
